Frequently Asked Questions

Do I need to create an account in order to use the mobile app or website?

To use the S.W.A.M.P website's functionality, a user must first register on the website. Per household, only one account is required: the name of the family's head, the names of the family members, phone numbers, and the address.

Do I need an internet connection when using the website?

Yes, an internet connection is needed to fully experience the features of the website.

How could I use the website if my internet connection is unstable?

Unfortunately, the website will be unavailable once the internet connection is disrepted.

Do I need to pay for subscriptions to fully enjoy the features of the website?

No, the website is free. To easily access the website, just scan the QR Code..
